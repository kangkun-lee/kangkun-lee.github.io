Welcome to [Hugo](https://gohugo.io/)! This is your very first post. Check [documentation](https://gohugo.io/documentation/) for more info. If you get any problems when using Hugo, you can find the answer in [troubleshooting](https://gohugo.io/troubleshooting/) or ask me on [GitHub](https://github.com/gohugoio/hugo/issues).

## Quick Start

### Create a new post

```bash
hugo new posts/my-new-post.md
```

More info: [Content Management](https://gohugo.io/content-management/)

### Run server

```bash
hugo server
```

More info: [Hugo Server](https://gohugo.io/commands/hugo_server/)

### Generate static files

```bash
hugo
```

More info: [Hugo Commands](https://gohugo.io/commands/)

### Deploy to GitHub Pages

GitHub Actions will automatically deploy your site when you push to the main branch.
